O CSS é um documento de estilização em cascata, criado em 1993, com sua primeira verção amplamente utilizada em 1996. A linguagem CSS permite definir regras que definem caracteristicas para os elementos HTML vomo cor, borda e comportamentos, além de animações e transições.

Ao longo dos anos o CSS evoluiu e hoje temos o CSS 1 2 e 3, cada uma das verções adicionou novos recursos que eram demandados no periodo

  1. CSS 1 - propriedades básicas de estilização e formatação, além de controle do posicionamento dos elementos HTML com display, propriedades float e position
  2. CSS 2 1998 - Adição de propriedades de animação e transição e melhoria do position
  3. CSS 3 2008 - incremento gradual de novas funcionalidades do CSS com animação e fereramentas de posicionamento como display:flex (2008) e grid (2013)

Uma regra CSS pode ser definida com a seguinte sintaxe

            Seletor{
                Propriedade1:Valor;
                Propriedade2:Valor;

                PropriedadeN:Valor;
            }

Podemos incluir codigo CSS de 3 formas diferentes

CSS externo
Quando definimos as regras CSS a serem utilizadas por meio da um arquivo externo .css, vinculando ele ao HTML por meio da tag <link> no head do documento HTML
CSS incorporado "no documento"
Quando definimos as regras CSS por meio da tag <style>
CSS inline
Quando definimos codigo CSS de estilização no próprio elemento HTML como: Texto de cor vermelha com CSS inline
