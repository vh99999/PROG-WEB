Lista com comandos basicos do git

  • git init - Cria um repositorio em uma pasta vázia
  • git add - Adiciona modificações do wk no stage
  • git status - Mostra o estado atual do repositorio , modificações arquivos versionados
  • git commit - Cria um commit no repositorio LOCAL
  • git push - Envia as modificações do repositorio LOCAL para o remoto
  • git log --oneline --all - Exibe todos os commit de forma resumida
  • git branch - Lista as branches do repositorio
  • git checkout - Muda de branch
  • git checkout -b <nome-da-branch> - Cria e muda para uma nova branch
