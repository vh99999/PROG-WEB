Lista com comandos basicos do git

  • git init - Cria um repositorio em uma pasta vázia
  • add - Adiciona modificações do wk no stage
  • status - Mostra o estado atual do repositorio , modificações arquivos versionados
  • commit - Cria um commit no repositorio LOCAL
  • push - Envia as modificações do repositorio LOCAL para o remoto
  • log --oneline --all - Exibe todos os commit de forma resumida
  • branch - Lista as branches do repositorio
  • checkout - Muda de branch
  • checkout -b <nome-da-branch> - Cria e muda para uma nova branch
